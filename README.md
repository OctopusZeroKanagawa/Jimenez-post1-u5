# Gestión de Tareas - Programación Web

Aplicación web desarrollada en Java utilizando Servlets y JSP como parte de la Unidad 5 de Programación Web. El proyecto implementa una gestión básica de tareas en memoria, permitiendo listar, agregar y eliminar tareas mediante peticiones HTTP GET y POST.

La aplicación utiliza el patrón Post/Redirect/Get (PRG) para evitar el reenvío de formularios al recargar la página y aplica validación en el servidor para el manejo seguro de los datos enviados desde formularios HTML.

## Tecnologías utilizadas

- Java 17
- Jakarta Servlet API 6.0
- JSP + JSTL
- Apache Tomcat 10
- Maven
- IntelliJ IDEA

## Ejecución del proyecto
1. Clonar el repositorio.
2. Abrir el proyecto en IntelliJ IDEA.
3. Configurar Apache Tomcat 10.
4. Ejecutar el artefacto war exploded.
5. Acceder desde el navegador a: http://localhost:8080/gestion-tareas/tareas
